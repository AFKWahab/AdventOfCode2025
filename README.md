# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025) written in Rust.

### Run a specific day and part:
On windwos:
```powershell
# Day 1, Part 1
Get-Content inputs/day01.txt | cargo run --bin day01_part1

# Day 1, Part 2
Get-Content inputs/day01.txt | cargo run --bin day01_part2
```

```powershell
Get-Content inputs/day01.txt | cargo run --release --bin day01_part1
```
On mac:
```bash
cat inputs/dayX.txt | cargo run --bin rust_script.rs
```
